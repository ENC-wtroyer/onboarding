# Workflow-Based Client Onboarding Checklist

### Client Overview
- [ ] Business information (locations, hours, contacts)
- [ ] Org chart (if applicable)
- [ ] Key decision makers & liason info
- [ ] Communication preferences & additional quirks
- [ ] Existing vendor relationships

### Credentials & Access Collection

- [ ] Domain admin accounts
- [ ] Service accounts documentation
